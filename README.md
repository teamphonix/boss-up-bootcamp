# Boss Up Bootcamp

Landing page skeleton for Boss Up Bootcamp, a hands-on AI entrepreneurship and digital creation program under The H.I.P.H.O.P. Academy.

## Local development

```bash
npm run serve
```

Then open:

```text
http://127.0.0.1:4175/
```

## Audit

```bash
npm run audit
```

## Project direction

- 20-seat bootcamp waitlist / registration page
- Showcase founder projects: music, graphics, videos, websites, apps
- Workflow-based AI creation curriculum
- Style direction: gritty black/white/yellow hip-hop flyer energy with purple, blue, green, and pink AI/creative accents
