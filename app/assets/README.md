# Asset Manifest

This folder structure is prepared so real Boss Up Bootcamp assets can be dropped in later without changing the site architecture.

## Folders

```text
app/assets/generated/   AI-generated hero/background/abstract assets
app/assets/music/       cover art, audio artwork, song/project visuals
app/assets/graphics/    flyers, ads, quinceañera designs, promo graphics
app/assets/videos/      video thumbnails or local video files if needed
app/assets/screenshots/ website/app screenshots and mockups
app/assets/logos/       H.I.P.H.O.P. Academy, Boss Up, partner logos
```

## Current handmade/SVG placeholders

```text
app/assets/hero-abstract.svg
app/assets/grit-texture.svg
app/assets/twenty-seats-sticker.svg
```

## Future form/backend options

The waitlist form is currently a placeholder. Good connection options:

- Google Forms: fastest/simple phone-friendly collection
- Airtable: good for lead tracking and tags
- Supabase: best if this becomes a larger web app
- Mailchimp/Brevo: best if email updates are the priority

## Replacement rule

Keep the current SVG assets as fallbacks even after AI images are generated. They are lightweight, fast, and useful for overlays/textures.
