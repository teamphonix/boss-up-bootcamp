- Paragraph font: Roboto
- Heading font: Roboto Bold

## Boss Up Bootcamp adaptation

The Boss Up Bootcamp page should feel like a direct extension of The H.I.P.H.O.P. Academy while staying polished and adult:

- Use black as the main site background.
- Use cyan as the Academy identity accent, not as a full-page flood.
- Use gold for CTAs, availability, and key program markers.
- Use Roboto/Roboto Bold for consistency with the mock site.
- Keep dropdown navigation and carousel presentation for a professional program-site feel.
- Use dark glass panels, thin cyan borders, and controlled gold highlights.
- Avoid childish rainbow/graffiti overload; keep hip-hop energy through contrast, rhythm, naming, and texture.
