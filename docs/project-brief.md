# Boss Up Bootcamp Website Skeleton Brief

Boss Up Bootcamp is a hands-on AI entrepreneurship and digital creation program under The H.I.P.H.O.P. Academy — How I Put Harmony On Pain.

## Mission

Teach ordinary people how to use AI tools to build businesses, brands, music, websites, apps, graphics, videos, and marketing campaigns without needing years of technical experience.

The site should make visitors think:

> If he built all of this using AI, maybe I can too.

## Primary Website Goals

1. Advertise Boss Up Bootcamp.
2. Collect waitlist/registration interest for the first session.
3. Promote limited seating: 20 seats per session.
4. Showcase Jonti's existing AI-powered work.
5. Act as a digital extension of The H.I.P.H.O.P. Academy.
6. Provide a foundation where future student projects can be added.

## Primary CTA

- Join The Waitlist
- Limited spots available
- 20 seats per session
- First session coming soon
- Register today to receive updates
